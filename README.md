# NeuroDriveCar

NeuroDriveCar is a browser-based autonomous driving simulation built with modern web technologies and a custom 2D engine architecture.

The project focuses on:

- Vehicle physics
- Sensor simulation
- Raycasting
- Neural networks
- Evolutionary learning
- Traffic simulation
- Real-time visualization

Unlike game-engine-based approaches, NeuroDriveCar is intentionally built with:

- TypeScript
- Vite
- HTML5 Canvas 2D
- Custom rendering and simulation systems

This allows full control over:

- Physics
- AI behavior
- Rendering
- Collision detection
- Sensor systems
- Evolution algorithms

---

# Vision

The goal of the project is to progressively evolve from:

```txt
Simple autonomous highway simulation
```

into:

```txt
A complete autonomous-driving sandbox with procedural roads,
traffic systems, intersections, city environments,
and emergent AI behavior.
```

# Core Features (Stage 1)

## Simulation

- Infinite 3-lane highway
- AI traffic vehicles
- Real-time collision detection
- Multiple simultaneous training cars
- Evolutionary learning cycles

## Vehicle Physics

- Acceleration
- Friction
- Steering
- Rotation-based movement
- Vehicle inertia

## Sensors

- Raycast-based distance sensors
- Road boundary detection
- Traffic detection
- Real-time sensor visualization

## Artificial Intelligence

- Feed-forward neural network
- Sensor input processing
- Mutation-based evolution
- Brain persistence using localStorage

- Real-time neural network visualizer
- Simulation HUD
- Camera tracking
- Best-car highlighting
- Sensor rendering

# Project Philosophy

This project intentionally avoids heavy game engines.

The objective is:

- Learn how the systems work internally
- Keep full architectural control
- Build simulation systems from scratch
- Create an extensible experimentation platform
